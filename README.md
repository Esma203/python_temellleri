Proje: python_temellleri

Bu küçük repo, `numbers.py` içinde basit bir maaş hesaplama örneği içerir.

Yerel olarak başlatıldı ve commit edildi. GitHub'a göndermek için aşağıdaki adımları kullanın:

1) GitHub CLI (önerilen):
   gh auth login
   gh repo create <kullanici-adi>/python_temellleri --public --source=. --push

2) Elle (PAT kullanarak):
   git remote add origin https://github.com/<kullanici-adi>/python_temellleri.git
   git branch -M main
   git push -u origin main
